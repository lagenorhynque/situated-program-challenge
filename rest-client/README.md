# rest-client

REST client implementation with Clojure.

## Usage

- with Leiningen:

```sh
lein uberjar
java -jar target/uberjar/rest-client-0.1.0-SNAPSHOT-standalone.jar [args]
```

OR

```sh
lein run [args]
```

- with [clj](https://clojure.org/guides/deps_and_cli) command:

```sh
clj -m rest-client.core [args]
```
